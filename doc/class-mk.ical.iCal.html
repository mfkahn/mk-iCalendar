Class iCal

A simple iCal parser.

http://github.com/mfkahn/mk-iCalendar based on http://github.com/fangel/SG-iCalendar

Roadmap: - Finish FREQUENCY-parsing. - Add API for recurring events

A simple example:

<?php
use mk\ical\iCal;
$ical = new iCal("http://example.com/calendar.ics");
foreach ( $ical->getEvents() As $event ) {
  // Do stuff with the event $event
}
?>
